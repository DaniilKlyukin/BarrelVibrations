# BarrelVibrations

**BarrelVibrations** — это программный комплекс для высокоточного имитационного моделирования физических процессов в артиллерийских системах. Приложение позволяет рассчитывать внутреннюю и внешнюю баллистику, тепловое состояние ствола и его динамические колебания при стрельбе сериями.

## 🚀 Ключевые возможности

*   **Внутренняя баллистика:** Расчет термодинамических параметров (давление, температура, плотность газа) с учетом различных типов порохов (трубчатые, зерненые) и геометрии каморы.
*   **Теплофизика:** Моделирование распределения температурных полей по толщине стенки ствола методом конечных разностей.
*   **Механика и НДС:** Расчет напряженно-деформированного состояния ствола (МКЭ) и решение задачи продольно-поперечных колебаний.
*   **Внешняя баллистика:** Расчет траектории полета снаряда (6 степеней свободы) с учетом деривации, ветра и влияния вибраций дульного среза на начальные условия вылета.
*   **Оптимизация геометрии:** Модуль автоматического подбора параметров ствола для минимизации массы при сохранении кучности стрельбы (методы Нелдера-Мида, Хука-Дживса).
*   **Идентификация параметров:** Обратная задача — уточнение характеристик пороха на основе экспериментальных данных.

## 🛠 Технологический стек

*   **Язык:** C# (.NET)
*   **UI & Визуализация:** 
    *   WinForms (интерфейс управления).
    *   **SharpGL (OpenGL):** Интерактивная 3D-визуализация геометрии ствола, снаряда и результатов расчетов (цветовые карты напряжений/температур).
    *   **ScottPlot:** Динамическое построение графиков физических величин.
*   **Математика и алгоритмы:**
    *   **MathNet.Numerics:** Решение систем линейных уравнений и интерполяция.
    *   **Численное интегрирование:** Методы Рунге-Кутты (4 и 6 порядка), Адамса (5 порядка) для решения систем ОДУ.
    *   **GMSH:** Интеграция с библиотекой генерации конечно-элементных сеток.
*   **Архитектура:** 
    *   Модульная структура солверов (Pattern Strategy).
    *   Многопоточные вычисления (`Parallel.For`, `TPL`) для ускорения тяжелых матричных операций.
    *   Сериализация сложных проектов в JSON (Newtonsoft.Json).

## 📐 Математический аппарат
В проекте реализованы:
1.  **FEM (Метод конечных элементов):** построение матриц жесткости и масс для различных типов элементов (Triangle, Quad).
2.  **Сплайн-интерполяция:** использование кубических сплайнов для аппроксимации характеристик материалов и таблиц метеоданных.
3.  **Аэродинамика:** расчет коэффициентов лобового сопротивления на основе регрессионных моделей.
